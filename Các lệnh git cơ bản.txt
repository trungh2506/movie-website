git clone <https://name-of-the-repository-link> : tải repository về máy
git status 							: xem thông tin trạng thái của repository
git add 	<file>						: thêm 1 file vào repository
git commit -m "commit message"				: tin nhắn về hành động đã làm (vd: git commit -m "add index.html file")
git push -u origin master					: đẩy file lên github
git pull 								: đồng bộ code từ github về máy


b1: Tạo 1 init trên máy (nếu chưa có)
	git init 
b2: Cấu hình cho git(user.email, user.name)
	vd: 	git config --global user.email |trung@gmail.com|
	vd:	git config --global user.name |trungh2506|
b3: Tải repository về máy
	vd: git clone https://github.com/trungh2506/movie-website.git
b4: Viết code
	Xem các file chưa được add
	git status
b5: Thêm file
	git add <file>
b6: Commit
	git commit -m "message"
b7: Đẩy code lên github
	git push -u origin master
b8: Đẩy code trên github về máy (nếu là sửa code)
	git pull
